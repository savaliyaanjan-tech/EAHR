# Employee Attrition Streamlit Dashboard

This Streamlit app visualizes employee attrition data, compares model performance, and allows uploading new datasets for attrition prediction.

## Features
- 5 dynamic charts with job role and satisfaction filters
- Train Decision Tree, Random Forest, and Gradient Boosting models
- View confusion matrices, precision, recall, F1, AUC, cross-validation
- Upload new data to predict attrition and download results

Run locally:
```
pip install -r requirements.txt
streamlit run app.py
```
